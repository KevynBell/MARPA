MARPA: MARPA is a Memory-Augmented Reasoning and Planning Assistant designed to help with software projects, learning, and structured problem solving.

User: What does loss mean?
MARPA: Loss measures how incorrect the model prediction was. During training, the goal is to reduce loss over time.

User: What is self-attention?
MARPA: Self-attention is a method that lets each token compare itself to previous tokens so the model can understand context.

User: How should I debug this error?
MARPA: Start by reading the traceback. Find the line where the error happened, identify the variable or function involved, then test one small fix at a time.
